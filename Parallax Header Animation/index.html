Desktop Support: Firefox, Chrome and IE, it even has basic support for IE7 (The images display and work but they need to be HUGE as backgrounds won't stretch).

Mobile Support: Chrome - OK, Android Browser - Broken, iOS - Not Tested.

Not responsive but support shouldn't be hard to put in.
